# 🎀 Cute Girl 2D Unity Project

<img src="Assets/Sprites/CuteGirl.png" width="256" alt="可爱卡通女孩预览图">

一个可爱卡通风的Unity 2D项目，包含萌萌的卡通女孩角色和完整的游戏框架。

</div>

## ✨ 项目特色

- 🎨 **可爱卡通风美术风格** - 粉色系可爱卡通女孩形象
- 🎮 **完整2D角色控制器** - 支持移动、跳跃、翻转
- 📷 **平滑相机跟随** - 自动跟随角色移动
- 💖 **精心设计的代码结构** - 清晰的C#脚本架构

## 📁 项目结构

```
CuteGirl2DProject/
├── Assets/
│   ├── Scripts/          # C#脚本文件
│   │   ├── CuteGirlController.cs    # 角色控制器
│   │   └── CameraFollow.cs          # 相机跟随
│   ├── Sprites/          # 2D精灵资源
│   │   ├── CuteGirl.svg            # 矢量图源文件
│   │   └── CuteGirl.png            # PNG预览图
│   ├── Scenes/           # 场景文件
│   ├── Prefabs/          # 预制体
│   └── Materials/        # 材质
├── Packages/             # Unity包配置
└── ProjectSettings/      # 项目设置
```

## 🚀 快速开始

1. 使用Unity 2022.3 LTS或更高版本打开项目
2. 打开 `MainScene` 场景
3. 点击Play按钮运行！

## 🎮 操作说明

- **← → / A D** - 左右移动
- **Space / W / ↑** - 跳跃
